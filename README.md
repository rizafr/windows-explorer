# windows-explorer
A full-stack Windows Explorer-like web application built as a Bun monorepo.


## Tech Stack

| Layer | Technology |
|-------|-----------|
| Runtime | [Bun](https://bun.sh) |
| Backend | [Elysia](https://elysiajs.com) + TypeScript |
| Database | PostgreSQL + [Drizzle ORM](https://orm.drizzle.team) |
| Frontend | [Vue 3](https://vuejs.org) + Composition API + [Pinia](https://pinia.vuejs.org) |
| Testing | Bun test (backend) · Vitest (frontend) · Playwright (E2E) |
